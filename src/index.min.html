Security, Perfomance, and Simple

Neo OS is a Linux distro with an emphasis on security, performance, and ease of use.
As much as possible Neo OS uses components built in memory safe programming languages(Go& Rust)
to improve safety while guaranteeing performance.

What's Special About Neo OS?

MesaLink

Memory Safety

MesaLink is impervious to bugs like Heartbleed and buffer overflows becuse it is written in the memory-safe language of Rust.

Modern Ciphersuites

MesaLink uses the best ciphersuites including AES-GCM, Chacha20Poly1305, and elliptic-curve key exchange with perfect forward secrecy.

TLS 1.3

Eight years since TLS 1.2, the faster and more secure TLS standard, is now in Rustls and MesaLink.

Blazing Fast

X25519 key exchange,AES-NI support, no language runtime like Java/Go. MesaLink runs at full speed on your metal.

Ion Shell

Memory Safety

The Ion Shell, written in Rust eliminates the possibilities of a ShellShock-like vulnerability.

Modern shell

Ion is a modern system shell that features a simple, yet powerful, syntax.

Speed

Performance that exceeds that of Dash(which itself is much faster than Bash), when taking advantage of Ion's features.

Omni Package Manager

Rust Powered

Developing a package manager in rust greatly increases the overall quality and security of the package manager while making deveolpment easier due to it's excellent build system.

Performance

Omni uses zstd compression while also downloading, unpacking, and installing packages in parallel and asynchronously. This means less time waiting on installs and updates and more time getting stuff done.

OUR

The Omni User Repository allows users to easily install a large variety of different packages that would not otherwise be available for via a traditional package manager(i.e. software that not avialable for distribution).

More Options

While Omni ships binaries by default you can also build packages with custom tweaks for your system.
